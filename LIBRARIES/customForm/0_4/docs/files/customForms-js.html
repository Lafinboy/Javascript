cstmForm

Custom Forms is a Jquery Pluggin that will make easier to customize form fields by creating flexible custom form elements.

FORM FIELDS SUPPORT

  • Select
  • Checkbox
  • Radio
  • File uploader

USAGE

All Form elements:

$('form').cstmForm()

Select:

$('select').cstmForm()

Upload:

$('input[type=file]').cstmForm()

Radio:

$('input[type=radio]').cstmForm()

Radio, checkbox, select:

$('input[type=radio], input[type=checkbox], select').cstmForm()

DEFAULTS

settingsDefault options
settings =
{
    customEle      : 'a',
    containerEle   : 'div',
    classPrefix    : 'custom-',
    autoHide       : 1, // Auto hide the stylized elements
    active         : 0, // Load all custom form modules
    select: {
        active: 1,
    },
    radio: {
        active: 1,
    },
    checkbox: {
        active: 1,
    },
    file: {
        active: 1,
    },
    text: {
        active: 1,
        prefix: 0,
        suffix: 0,
        prefix_txt: "Please enter ",
        suffix_txt: "..",
        customText:
        {
            // example
            // name: "Enter your name"
        },
        callback: function(){}
    }
}

PARAMETERS

optionsobject will overwrite default
  • Example :
$('form').cstmForm({ customEle: 'span' });
